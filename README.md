Instructivo de uso - inventario.py

Abrir el archivo
Abre el archivo inventario.py en tu editor de Python (por ejemplo, IDLE o VS Code).

Ejecutar el programa
Presiona Ejecutar o usa la terminal con:

python inventario.py

Ingresar los datos
El programa te pedirá:

Nombre del producto

Precio del producto

Cantidad del producto

Escribe cada dato y presiona Enter.

Ver el resultado
El programa mostrará en pantalla algo como:

Producto: Lápiz | Precio: 500 | Cantidad: 3 | Total: 1500

Fin del programa
